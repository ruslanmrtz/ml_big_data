1. Используя специализированные библиотеки работы с геоданными, загрузите исходные
файлы в векторном ГИС-формате – 3 слоя.
2. Соедините 3 слоя, получите полную карту. Визуализируйте ее.

Есть такие файлы
1. Слой etalons.shp содержить информацию об основных характеристиках лесов для формирования обучающей выборки
2. Слой forestry_bounds.shp содержит границы тестового участка
3. Слой etalons_clearcuts.shp содержит информацию о выявленных за ревизионный период (2021-2023 гг.) лесоизменениях
Кроме этого для каждого слоя есть файлы .cpg, .dbf, .prj, .shx.
